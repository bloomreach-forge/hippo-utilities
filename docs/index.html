Introduction

What is Hippo Utilities forge project

The Hippo Utilities forge project contains generic code for reusage, in order to avoid code being copied from project to project.

Code base on GitHub

The code is on GitHub at https://github.com/bloomreach-forge/hippo-utilities/.

Feature overview

There are four artifacts available, targeted for CMS, repository and HST, plus one common artifact.

hippo-utilities-commons
  • Annotation Utility
  • Generics Utility
  • Hippo Utility
  • Repository Utility
  • Simple FTP Client
  • JCR Mockup to create a node structure based on system view XML
hippo-utilities-hst
  • Component utility methods (e.g. get parameter with a default)
  • Base components reading id and class config for div, also Container component reading HST subcomponents dynamically, RedirectComponent and ForwardComponent
  • Cookie support check utility
  • Paging
  • XML Stripper
  • Annotation based reading of repository nodes, for example for configuration nodes outside of /content/documents
hippo-utilities-repository
  • Groovy Updater Execution Scheduler Job
