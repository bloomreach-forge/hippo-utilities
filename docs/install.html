How to install the Hippo Utilities modules

Maven 2 Forge Repository

Add this repository to your root pom.xml (if not already present in a parent pom):

  <repository>
    <id>hippo-forge</id>
    <name>Hippo Forge maven 2 repository.</name>
    <url>http://maven.onehippo.com/maven2-forge/</url>
    <snapshots>
      <enabled>false</enabled>
    </snapshots>
    <releases>
      <updatePolicy>never</updatePolicy>
    </releases>
    <layout>default</layout>
  </repository>

Installation of HST module

Assuming that the version is managed in the parent pom, add this dependency to the pom.xml of your site module:

<dependency>
  <groupId>org.onehippo.forge.utilities</groupId>
  <artifactId>hippo-utilities-hst</artifactId>
</dependency>
